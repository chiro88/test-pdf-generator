# picker_cmc (v0)

No-truth PDF figure/table/common-region detector + a local web editor for operator
correction, with a downstream object-package export. This `picker_cmc_v1/` tree is
the product; the surrounding RTM factory generates the golden fixtures the detector
is regression-tested against.

Released as **`picker-cmc-v0-rc1`**. See `../docs/product/RC_FINAL_SIGNOFF.md`.

## Install

```bash
# from the repo root (real_tracking_mock_v1/)
pip install -r requirements.txt          # PyMuPDF + PyYAML (server is stdlib)
```

## One-command check

```bash
cd picker_cmc_v1
python tools/run_product_e2e_smoke.py --pdf /path/to/input.pdf --workdir /tmp/e2e --json
```

## Product flow

```
setup-yaml-v0
  → detector run (detector-output-v0)
  → editor-save-manifest-v0
  → web viewer/editor (bbox edit / ruler / save / save-as)
  → reopen
  → edited review export (overlays + crops)
  → downstream-package-v0 (objects.jsonl + per-object crops)
```

All coordinates are PDF points, top-left origin (`[x0, y0, x1, y1]`, no axis flip).

## Layout

| dir | what |
|---|---|
| `detector/` | no-truth detector (extraction, caption anchors, region inference, common regions) |
| `detector_output/` | `detector-output-v0` schema/writer/validator |
| `setup/` | `setup-yaml-v0` schema/template/loader/validator |
| `editor_manifest/` | `editor-save-manifest-v0` schema/writer/validator |
| `web_editor/` | stdlib web server + static viewer/editor + export |
| `downstream_package/` | `downstream-package-v0` exporter/validator |
| `tools/` | CLIs (see below; all support `--help`) |
| `tests/fixtures/rtm_frozen/` | committed golden set (49 cases) for detector regression |
| `tests/fixtures/rtm_factory/` | RTM factory (regenerates `rtm_gallery/`) |

## CLIs (`tools/`, run from `picker_cmc_v1/`)

| tool | purpose |
|---|---|
| `detect_pdf.py` | detector on one PDF → detector-output-v0 |
| `run_detector_on_pdf.py` | real-PDF review harness (overlays/crops/manifest) |
| `run_detector_on_rtm.py` | RTM frozen regression runner |
| `make_setup_template.py` | emit a setup-yaml-v0 template |
| `run_detector_with_setup.py` | detector run from a setup file |
| `run_web_editor.py` | launch the local web editor (`--run-dir` or `--setup`) |
| `export_editor_manifest_artifacts.py` | post-edit overlays/crops from the edited manifest |
| `export_downstream_package.py` | downstream-package-v0 export |
| `summarize_review_feedback.py` | aggregate operator review feedback |
| `validate_editor_manifest.py` | validate an editor-save-manifest |
| `run_product_e2e_smoke.py` | end-to-end product smoke |

## Tests + regression

```bash
# from the repo root
python -m pytest tests picker_cmc_v1 -q
# RTM regression (from picker_cmc_v1/)
python tools/run_detector_on_rtm.py --detector-cmd "python tools/detect_pdf.py --pdf {pdf} --out {out}" --json
```

Baseline: pytest passing; RTM cases 47/49, objects 100/100, regions 206/210 (the 4
region failures are the documented rotated/morph/image-like watermark limitation).

## RTM factory (golden fixtures)

```bash
cd tests/fixtures/rtm_factory && python generate.py   # -> ../rtm_gallery/ (git-ignored)
```

The gallery includes sequence examples (figure/table/text under common headers/
footers/watermarks), e.g. `exp_hfwm_*` = a figure/table/text sequence over a full
header + footer + watermark base. `rtm_frozen/` is the human-approved subset used
for regression and is **not** regenerated by this command.

## Scope (v0)

Geometry/structure only — no semantic waveform/diagram/table interpretation and no
LLM calls. The web editor is local single-user (no auth/DB/cloud). Real-PDF results
are operator visual-review proposals, not golden truth.
